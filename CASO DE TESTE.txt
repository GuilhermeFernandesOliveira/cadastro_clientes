CASO DE TESTE


Teste 1 - Opção inválida

Passos:
1. Abrir o sistema
2. Inserir uma opção que não esta aparecendo no menu

Resultado esperado:
Mensagem: "Opção inválida, tente novamente:"


Teste 2 - Cadastro válido

Passos:
1. Abrir o sistema
2. Escolher opção cadastrar
3. Inserir nome válido
4. Inserir idade válida

Resultado esperado:
Mensagem: "Cliente cadastrado com sucesso!"


Teste 3 - Listar/Editar sem clientes

Passos:
1. Abrir o sistema
2. Escolher Listar (ou Editar)

Resultado esperado:
Mensagem: "Nenhum cliente cadastrado"


Teste 4 - Idade negativa no Cadastro

Passos:
1. Abrir o sistema
2. Cadastrar cliente
3. Inserir idade -5

Resultado esperado:
Mensagem: "Idade negativa, tente novamente:"


Teste 5 - Opção inexistente/inválida na seção Editar

Passos:
1. Abrir o sistema
2. Cadastrar cliente
3. Escolher a opção Editar
4. Inserir opção 7 (Opção inexistente/inválida)

Resultado esperado:
Mensagem: "Opção inválida, tente novamente:"


Teste 6 - Editar cadastro

Passos:
1. Abrir o sistema
2. Cadastrar cliente
3. Escolher a opção Editar
4. Escolher a opção de Editar Cadastro
5. Escolher o cadastro que quer editar de acordo com o ID
6. Editar as informações 

Resultado esperado:
Mensagem: "Edição de cadastro com sucesso!"


Teste 7 - ID inexistente/negativo na seção de Editar as informações de um Cadastro ou na seção de Excluir um Cadastro.

Passos:
1. Abrir o sistema
2. Cadastrar Cliente
3. Escolher a opção Editar
4. Escolher a opção de Editar as informações de um Cadastro (Ou Excluir um Cadastro)
5. Inserir ID -2 (ou um ID que não existe)

Resultado esperado:
Mensagem: "ID inválido, tente novamente:"


Teste 8 - Excluir cadastro

Passos:
1. Abrir o sistema
2. Cadastrar Cliente
3. Escolher a opção Editar
4. Escolher a opção Excluir um Cadastro
5. Escolher o ID correspondente ao qual Cadastro gostaria de Excluir

Resultado esperado:
Mensagem: "Exclusão de Cadastro com sucesso!"


Teste 9 - Voltar ao menu

Passos:
1. Abrir o sistema
2. Cadastrar um Cliente
3. Escolher a opção Editar
4. Escolher a opção Voltar ao menu

Resultado esperado:
Voltar ao menu inicial novamente


Teste 10 - Enviar uma Letra/Palavra/Frase ao invés de um Número

Passos:
1. Abrir o sistema
2. Inserir a letra H

Resultado esperado:
Mensagem: "Entrada inválida, insira um numero:"

(Este teste se aplica em todas as ocasiões em que se pede um número para ser inserido)
